Decription

The basemap is obtained from Supervisor District boundaries. Each district in the base map has its supdist and neighbouWe use a geospatial data visualization technique to visualize data from the "311 Tree Maintenance Requests" dataset. To be more specific, This is a non-proportional symbol map that colored by column "Status", which has pink represents open status and lightblue shows closed status. Each district in the base map shows its supdist and supname, while each point represents a tree maintenance record and shows additional information such as neighbourhood, request details and supervisor district. It is obeserved to conclude that (blue)closed case is the majority, and it also shows where the most trees are planted. This data visualization can provide useful infomation to goverment authority about making decision in tree maintenance.
